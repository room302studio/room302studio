# IssueBuilder

IssueBuilder transforms written Statement of Work documents into GitHub issues using AI, making project kickoffs smoother and more systematic.

Every time we start a new project at Room 302 Studio, we go through the same process: reading through the SOW, breaking it into tasks, and manually creating dozens of GitHub issues. IssueBuilder was born from the frustration of doing this repeatedly.

Paste or upload a document, and IssueBuilder breaks it down into well-structured, actionable GitHub issues — complete with descriptions, labels, and suggested milestones. It speeds up a process that used to take hours into minutes.
